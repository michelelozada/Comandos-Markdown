Listando itens via linguagem Markdown    
Repositório: Comandos Markdown  
GitHub: @michelelozada    
  
  
**1. Lista numerada (aka listas ordenadas):**  
. marcação  
```
Lista:
   1. Primeiro item 
   1. Segundo item 
   1. Terceiro item 
```

. output  
Lista:
   1. Primeiro item 
   2. Segundo item 
   3. Terceiro item    
  
   
**2. Lista de marcadores (aka listas não ordenadas):**  
. marcação    
```
Lista:
   * Primeiro item 
   * Segundo item
   * Terceito item
```
ou

```
Lista:
   - Primeiro item 
   - Segundo item
   - Terceiro item
```
  
. output    
Lista:
   * Primeiro item 
   * Segundo item
   * Terceiro item
     
  
**3. Listas com subitens:**  
*(Obs: Apenas para listas de marcadores)*  

. marcação:    
```
Lista:
- Primeiro item 
	- Primeiro subitem
	- Segundo subtitem
- Segundo item 
	- Primeiro subitem
	- Segundo subtitem
- Terceiro item 
```
  
. output:  
Lista:
- Primeiro item 
	- Primeiro subitem
	- Segundo subtitem
- Segundo item 
	- Primeiro subitem
	- Segundo subtitem
- Terceiro item
